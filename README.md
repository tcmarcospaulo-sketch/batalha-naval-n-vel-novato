# Desafio Nível Novato - Batalha Naval em C

Este repositório contém a solução do desafio de posicionar navios em um tabuleiro 10x10 usando linguagem C.

- Tabuleiro representado por uma matriz 10x10.
- Dois navios de tamanho 3: um horizontal e um vertical.
- Validação de limites do tabuleiro e sobreposição.
- Exibição do tabuleiro no console com 0 (água) e 3 (navios).
